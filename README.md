Ruby crypto examples
====================

## Supported:

* AES (128 / 256)
* SSHA
* SHA1
* SHA2
* SHA3 ( 256 / 384 / 512 )
* MD5
* MD4
* Blowfish (@dlagevin)
* File sha1/md5 validation
* LDAP ssha password generation
